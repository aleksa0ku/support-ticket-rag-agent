# Sharing Files and Folders with Links

## Creating a share link

Right-click any file or folder and choose "Copy Cloudbox link" (desktop) or
tap the share icon (mobile/web). Anyone with the link can view the item
using the default settings.

## Link permissions

From "Manage link" you can set:

- **Can view** or **Can edit** (folders only on Plus and above).
- **Password protection** — requires viewers to enter a password you set.
- **Expiration date** — the link stops working automatically after this
  date.
- **Download disabled** — viewers can preview but not download (Business
  plan only).

## Revoking a link

Go to Settings > Shared > Links, find the link, and click "Delete link."
This immediately invalidates it — anyone with the old URL gets an error.
Deleting a link does not delete the underlying file.

## Who can see my shared links?

Only people you send the link to (or who find it independently — links are
not indexed by search engines). Files aren't otherwise visible to anyone
outside your account unless explicitly shared.

## Link limits by plan

Free accounts can create up to 10 active links at a time. Plus, Family, and
Business plans have no limit on active links.
