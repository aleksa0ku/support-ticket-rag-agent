# Installing and Updating the Desktop App

## System requirements

- Windows 10 or later (64-bit)
- macOS 12 (Monterey) or later, Apple Silicon and Intel both supported
- At least 2 GB free disk space beyond your sync folder size for the local
  cache

## Installing

Download the installer from the Downloads page and run it. On macOS you may
need to approve the app in System Settings > Privacy & Security if
Gatekeeper blocks it on first launch.

## Updating

The desktop app checks for updates automatically and installs them in the
background; you'll be prompted to restart the app to finish. You can check
your current version and force a check from the tray/menu-bar icon >
Settings > About.

## Uninstalling

Uninstalling the app (via Add/Remove Programs on Windows or dragging to
Trash on macOS) stops syncing on that device but does not delete your files
from the cloud or from the local Cloudbox folder — those remain on disk as
regular files.

## Multiple accounts on one device

The desktop app supports signing into one account at a time per installed
instance. To sync two separate accounts on the same computer, install a
second, separately-configured instance using our multi-instance setup
guide, or use the web app for the second account.
